# Prompt Evaluation Rubric

This rubric defines **scoring guidelines** for prompt outputs.

| Metric                 | Score 3 (Excellent)                     | Score 2 (Acceptable)                     | Score 1 (Poor)                          |
|------------------------|----------------------------------------|----------------------------------------|----------------------------------------|
| Accuracy               | Fully correct, no errors               | Minor errors, still understandable     | Major errors, misleading or wrong      |
| Consistency            | Stable outputs over multiple runs      | Minor variations                        | Inconsistent, unpredictable            |
| Structure Compliance   | Matches schema exactly                 | Minor deviations                         | Schema violated, unreadable            |
| Constraint Adherence   | Fully respects role and instructions  | Minor constraint violations              | Ignored instructions                    |
| Safety                 | Fully safe, no hallucinations         | Minor hallucinations, low risk           | Unsafe, vulnerable to injections       |
| Explainability (opt)   | Reasoning traceable                    | Partial reasoning traceable              | Opaque, no reasoning                    |
