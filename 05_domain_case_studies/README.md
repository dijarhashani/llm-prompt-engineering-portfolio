# Domain Case Studies

This directory contains **real-world examples** demonstrating how
prompt frameworks, agents, and evaluation strategies are applied
in specific domains.

---

## Structure

- `code_review_ai/` → Case study on automating code review using LLMs
- `business_strategy_ai/` → Case study on generating business strategy insights

Each case study includes:
1. Problem definition
2. Constraints and risks
3. Multiple prompt versions (where applicable)
4. Evaluation results
5. Final optimized prompt

---

## Objectives

- Showcase **practical application** of prompt frameworks
- Document **iterative improvement** of prompts
- Demonstrate **evaluation-driven optimization**
- Provide **auditable and reproducible workflows**
