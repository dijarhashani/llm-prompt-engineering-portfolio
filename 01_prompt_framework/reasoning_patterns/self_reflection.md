# Self-Reflection Pattern

This pattern forces the model to **critique its own output**.

---

## Prompt Strategy

After generating an answer:
- Review for correctness
- Identify assumptions
- Correct mistakes

---

## Example Instruction

Review your previous output.
List any errors or uncertainties.
If found, provide a corrected version.

---

## Risks
- Infinite loops
- Over-correction

Use sparingly.
